== Description ==

Nosh is a responsive and modern looking theme which is ideal for travel bloggers, fashion bloggers, life style bloggers etc.It has a featured post section in front page along with 3 column grid layout for recent blog posts. The blog page displayes blog posts in 2 column layout with a dynamic sidebar. Header section also features a responsive drop down menu and social media profile icons. 

For more information about NOSH WordPress Theme, please visit to https://scratchtheweb.com/product/responsive-free-wordpress-theme-nosh/.

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Type in NOSH in the search form and press the 'Enter' key on your keyboard.
3. Click on the 'Activate' button to apply the theme on your site.
4. To set up front page and blog page, go to Pages -> Add New and create new pages titled: Home and Blog.
5. Navigate to Settings -> Reading and for the option "Your homepage displays", select "A static page" and set Home for Homepage and Blog for Posts page.
6. To enable social media icons on the Header, go to Appearence -> Customize > Theme Settings and enter the URL of your social media pages.
7. To setup Menus, go to Appearance -> Menus and create menus for Header Menu, Footer Menu 1 and Footer Menu 2.
